6. 특수 기호

bash -s


~ : 홈 디렉토리
$ : 변수의 값을 표출하기 위해 변수 이름 앞에 사용되는 기호
& : 백그라운드로 작업을 실행하기 위해 사용되는 기호
                    ex) sleep 60 &
# : 주석 기호
* : 와일드 카드 문자 >> *.sh
? : 와일드 카드 문자 >> ?.sh, a.sh b.sh c.sh
\ : 다음 문자 일반 문자화
( : 하위 쉘 시작
) : 하위 쉘 끝
! : not
/ : 디렉토리 경로 구분자
; : 쉘 명령 구분자
< : 입력 리다이렉션 기호. 파일로부터 입력값을 받음
> : 출력 리다이렉션 기호. 덮어쓰기의 의미
>> : 출력 리다이렉션 기호. 추가의 의미
`cmd` : 역따옴표, Grave, 명령 내 명령을 실행할 경우
| : 파이프 명령의 출력값을 다른 명령의 입력값으로 사용할 경우 사용
[ : 문자 집합의 시작을 나타냄
] : 문자 집합의 끝을 나타냄

[의심되는 부분만 디버깅 모드로 동작하기]
set -x 디버깅 시작점, set +x 디버깅 종료점
