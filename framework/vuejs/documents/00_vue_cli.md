# Vue CLI 시작하기

 - [VUE CLI](https://cli.vuejs.org/#getting-started)

## [ install ]

```sh
npm install @vue/cli
```

## Create Vue Project

 - legacy : `init`
 - now : `create`

```sh
npx vue create hello-world

# You can also create and manage projects using a graphical interface with the vue ui command:
npx vue ui
```
